# Technology Stack

## Core
- **Programming Language:** TypeScript
- **Frontend Framework:** Next.js (App Router)
- **Library:** React

## Backend & Database
- **Platform:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth (OTP/Passwordless)
- **Real-time:** Supabase Real-time for live leaderboard updates.

## UI & UX
- **Styling:** Tailwind CSS (utility-first styling).
- **Components:** Radix UI and Shadcn (accessible, headless UI primitives).
- **Animations:** Three.js and Framer Motion (deferred to later phase for immersive effects).

## State & Data Management
- **Forms:** TanStack Form for complex validation and team submissions.
- **Tables:** TanStack Table for flexible, high-performance leaderboard displays.
- **Context:** React Context API for global state like user authentication and round state.

## Integration & Tools
- **Data Scraping:** HackerRank integration via API/scraping routes.
- **Deployment:** Vercel (recommended for Next.js).
