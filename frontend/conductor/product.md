# Initial Concept
Pixtopia is a Pixar-themed tech competition platform built for the GDG club to host and manage their signature events. The website provides an immersive experience with high-quality animations and real-time interactive dashboards for participating teams.

# Product Guide

## Vision
To create a highly interactive, real-time competition hub that automates event management for GDG club organizers while providing an engaging, Pixar-themed experience for participants.

## Target Users
- **GDG Club Admins/Organizers:** Manage rounds, questions, submissions, and the overall event flow.
- **Participant Teams:** Compete in technical games, view questions, submit answers, and track their progress on the dashboard.
- **Spectators/Viewers:** Follow the competition and see the live leaderboard updates.

## Primary Goals
- **Efficient Round Management:** Streamline the process of hosting multiple technical games and questions.
- **Real-Time Leaderboard:** Provide instant, visually engaging score updates to keep competitors and viewers informed.
- **Data Integration:** Automate scoring and progress tracking using platforms like HackerRank and Supabase.
- **Secure Access:** Ensure teams can safely login and interact with their dedicated dashboard.

## Key Features
- **Interactive UI/UX:** An immersive Pixar-themed interface using Three.js and Framer Motion for high-quality animations.
- **Integrated Supabase Backend:** Robust backend for authentication, round state management, and real-time database updates.
- **Automated Scoring/Submissions:** Smooth processing of team submissions with automated scoring and integration with external platforms.
- **Dynamic Dashboard:** A central hub for teams to view their status, the current round details, and the global leaderboard.
